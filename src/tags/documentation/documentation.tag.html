Task Roulette
Task Roulette is not your tipical task manager but insted is a manager of tasks.

Task Roulette will choose the minimal set of tasks for you to do in a day, when tasks are done no more tasks are given for the day. Therefor task roulette is more suituble for hobbyts and personal use.
Task Manager
Because task roulette is meant to be simple and release the user from the task of task managing task manager will appear only once per week.

The main purpose of task managing is to clean the list of tasks from tasks alredy done or duplicated.
Tags
Tags can be added in tasks by using #tag-name, tags can be used to organize or tasks and also be used in sprints.

  • Task Examples:
    • Task Content: #lunch Japonese
    • Task Content: #lunch Chinise
    • Task Content: #lunch Portuguese
    • Task Content: #work #task-roulette Do better documention
    • Task Content: #work #task-roulette Make new release
Sprints
Sprints allow user to link a tag to a date.

When sprints are set task roulette will choose and manage a minimal set of tasks per day so that the sprint date is meet with no tasks with the sprint tag.
