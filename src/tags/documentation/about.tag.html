About
Task Roulette is a open source project developed by Filipe Vieira.

Task Roulette was developed with one purpose in mind, encourage you to progress in multiple projects, promoting all tasks in the same way, making sure that you don't neglate any project.
It also has a pisicological side on it, by selecting a limited number of tasks per day widhout overloading you with tasks and managing your tasks it will give you a good filling of completion each day, making you to come back next day.
Open Source
Task Roulette is an open source project, using many other open source projects.
